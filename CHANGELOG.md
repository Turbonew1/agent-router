# Changelog

All notable changes to agent-router.

## [1.0.0] - 2026-06-28

### Added
- Initial release
- Skill scanner (`scan-tools.sh`) — detects skills, agents, MCP tools
- Auto-categorization of 20+ tool categories
- Bilingual routing rules (English + Chinese)
- Priority-based routing logic
- Install script with one-command setup
- Compatible with `npx skills add`
- Plugin manifest (`.claude-plugin/`)

### Skills Supported
- 483+ skills auto-detected
- 62+ agents auto-detected
- MCP tools: mcp-video, playwright, github, memory, context7, exa, sequential-thinking
